== Installation ==

1. Upload 'learndash-favorite-content' folder to the '/wp-content/plugins/' directory or upload the learndash-favorite-content.zip through the WordPress admin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can configure it at LearnDash > Settings > Favorite Content
4. Activate your license
